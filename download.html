System requirements

What you need to run it.

Same baseline as Arch Linux. If your machine ran Arch, it'll run XyvorraOS.

Architecture x86_64 (64-bit)
RAM 512 MB min · 2 GB rec
Storage 2 GB min · 20 GB rec
Firmware UEFI or BIOS
Internet Required for install
USB drive 1 GB+ for flashing
Getting started

Once you have the ISO.

Three steps from ISO to running system. The installer handles the rest.

01
Get the ISO

Request early access via Discord. You'll get a direct download link to the v1.0.2 ISO once your request is approved.

Request access on Discord ↗
02
Flash to USB with Rufus

Open Rufus, select your USB drive, load the ISO, and write it. Takes about two minutes. Boot mode auto-detected.

Full guide on GitHub ↗
03
Boot and install

Boot from the USB, follow the first-boot wizard, and you're in. UEFI and BIOS both supported — old ships welcome.

Install guide on GitHub ↗
Flashing tool

Why Rufus?

Rufus — free USB flashing tool for Windows

Fast, reliable, and straightforward. Select your drive, load the ISO, click start. Works with both UEFI and BIOS boot targets and auto-detects the right partition scheme. Linux users can flash from the terminal using dd instead.
