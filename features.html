Custom Installer

Full UEFI and BIOS support — old hardware deserves love too. Guided partitioning, timezone and locale setup, all in one clean flow.

First-Boot Wizard

Walks you through theme selection, package choices, and user setup right from the first boot. No digging through configs on day one.

xyvorra-update

One command to pull OS updates, theme changes, scripts, and packages. Version checking, clean rollout, and rollbacks included.

$ xyvorra-update
→ Something broke?
$ xyvorra-rollback
