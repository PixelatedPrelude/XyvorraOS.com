What XyvorraOS actually does.

Shipped features, active builds, the full roadmap, and the big ideas — all in one place.

Visual identity

Four themes. Hand-crafted.

Not generated, not templated. Each palette feels like a different corner of the galaxy. Future theme contests coming — join Discord to participate.

Galactic Byzantium

Deep purple-black. Regal and mysterious.

Quantum Verdance

Bioluminescent green. High energy.

Void Pearl

Default. Deep navy. Built for long sessions.

Rosefire Eclipse

Blackout crimson. Not subtle.

🎨 Theme contests coming

Got a concept for a new theme? Ideas and fan designs are welcome in the Discord community. Winning designs may ship in a future release.

Roadmap

High-impact ideas incoming.

These are on the radar — some closer than others. Submit a feature request on GitHub to help prioritize.

🖥️

Mission Control Dashboard

System health cockpit: CPU/GPU/RAM/temps/network/battery in one HUD. Alerts panel. Quick actions: clear cache, restart services, update system.

◌ Planned
🤖

AI Command Bridge

Local assistant for terminal and system tasks — "optimize performance mode", "find large files", "fix broken package cache". Python backend + QML frontend.

◌ Planned
⚡

Profiles / Flight Modes

Toggle presets: Stealth (battery saver), Warp (max performance), Docked (balanced), Broadcast (streaming). Script CPU governor, compositor, network, services.

◌ Planned
📸

TimeShift+ Snapshot UX

Clean GUI for Btrfs/Timeshift: "Create restore point before updates", "Boot into previous snapshot", visual timeline restore.

◌ Planned
🚀

Space Terminal Suite

Themed terminal app with preconfigured Zsh/Fish prompt, command cards, split-pane monitoring, persistent project sessions.

◌ Planned
🔍

App Dock + Command Launcher

Fast launcher with fuzzy search and command palette style (VS Code-style, system-wide). Quick actions: Bluetooth, screenshot, VPN, audio sink.

◌ Planned
🛡️

Security Hardening

GUI firewall defaults (UFW frontend), automatic snapshot before risky operations, health checks service, signed update metadata and mirror fallback.

◌ Planned
🌐

Network Radar

Visual LAN scanner, connection quality monitor, DNS latency test — all in a themed QML interface.

◌ Planned
