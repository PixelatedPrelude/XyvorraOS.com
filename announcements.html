XyvorraOS v1.0.0 — Baseline Release

This is it. Version 1.0.0 is the baseline release of XyvorraOS — the starting line for everything that comes after. It's not perfect. Some things are rough. But it boots, it installs, it looks the way it's supposed to look, and it's out there.

If you're installing it, you're an early crew member. That means you'll see the rough edges first — and you'll get to watch this thing grow from here. File issues on GitHub when things break. It genuinely helps. Come hang out in Discord if you want to see what's being built next.

There will be many updates. Oh so so many.
